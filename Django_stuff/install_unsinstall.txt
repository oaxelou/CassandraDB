(Install) Οδηγίες εγκατάστασης της εφαρμογής:

Η ιστοσελίδα είναι γραμμένη σε Django 1.11. Για να αναπαράγει κάποιος το χτίσιμο της ιστοσελίδας μας πρέπει να ακολουθήσει τα εξής βήματα:
1)	Να δημιουργήσει ένα virtual environment όπου θα δημιουργήσει το Django project.
Linux: $> virtualenv -p python2 .
2)	Να ενεργοποιήσει το virtual environment.
Linux: $> source bin/activate
3)	Να σιγουρευτεί πως όλα τα πακέτα που χρειάζονται για την εφαρμογή μας είναι ήδη εγκατεστημένα στο virtual environment. Τα πακέτα που χρειάζονται είναι στο requirements.txt στο GitHub (https://github.com/oaxelou/CassandraDB/blob/master/Django_stuff/requirements.txt). Οπότε πρέπει να τρέξει την εντολή (με την προϋπόθεση πως το αρχείο requirements.txt βρίσκεται στο directory env:
Linux: $> pip install -r requirements.txt 
4)	Να κατεβάσει ολόκληρο το project directory (src) από το Github στον φάκελο env 
5)	Να σηκώσει τον django server. Δηλαδή, στο directory env\src:
Linux: $> python manage.py runserver
6)	Τέλος, για να τρέξει την εφαρμογή, πρέπει να πλοηγηθεί μέσω ενός browser στη διεύθυνση που αντιστοιχεί στον django server, δηλαδή στο localhost + django port: http://127.0.0.1:8000/

-------------------------------------------------------------------------------
(Uninstall) Οδηγίες απεγκατάστασης της εφαρμογής:
Μιας και η εφαρμογή είναι χτισμένη πάνω σε virtual environment, η διαγραφή της από το σύστημά μας είναι ιδιαίτερα εύκολη: απλά διαγράφουμε όλα τα περιεχόμενα του φακέλου μέσα στον οποίο φτιάξαμε το virtual environment. Με αυτόν τον τρόπο, διαγράφουμε ταυτόχρονα όλα τα αρχεία της εφαρμογής (που είναι στον υποφάκελο src) και όλα τα python packages που εγκαταστήσαμε με το pip (είναι όλα συγκεντρωμένα στον υποφάκελο Lib).
